# vue-quill-editor
Quill rich text editor component encapsulated with Vue3


# install

    npm install

# import into project:

```javascript
import VueQuillEditor from 'vue-quill-editor';
import 'vue-quill-editor/dist/vue-quill-editor.esm.css';

...

app.use(VueQuillEditor);

app.mount("#app-wrapper");

```
